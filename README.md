NestRTD
=======

C++ RTD client for Nest. Feeds Amibroker.
Also included is a backfill tool to import VWAP statistics / Nest Plus Data table 


Build
    Build using VC++ 2010 Express. ATL headers and libs taken from Windows Driver kit
